This is a simple water pump automation built around JSN-SR04T-2.0-ultrasonic-sensor.
this should also work with similar ultra sonic sensors as well



Components used
1. 16x2 lcd(i2c)
2. arduino nano
3. JSN-SR04T
4. 103 thermistor
5. resistors(10k)
6. relay module etc
